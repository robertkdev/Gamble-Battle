# Bundled Font Licenses

Gamble Battle bundles the following fonts for its user interface.

## Cinzel

- Upstream: <https://github.com/google/fonts/tree/main/ofl/cinzel>
- Bundled file: `assets/fonts/cinzel/Cinzel[wght].ttf`
- License: SIL Open Font License 1.1
- Bundled license: `assets/fonts/cinzel/OFL.txt`

The font is included unmodified. The wordmark and ceremonial display typography use variable weight instances produced by Godot at runtime.

## Source Sans 3

- Upstream: <https://github.com/adobe-fonts/source-sans>
- Bundled files: `SourceSans3-Regular.ttf`, `SourceSans3-Semibold.ttf`, and `SourceSans3-Bold.ttf`
- License: SIL Open Font License 1.1
- Bundled license: `assets/fonts/source_sans_3/LICENSE.md`

The font files are included unmodified. Source Sans 3 is the primary body and interface family.

These license copies must remain with redistributed builds that contain the font files.
